# DungeonRush
**Dungeon rush is a simple 2D platformer game made using Apollo.**

## Apollo 
**Apollo is a 2D game engine that implements ECS and has useful utility functions for 2D rendering and loading assets**

![DungeonRush](https://github.com/SauravBastola/DungeonRush2/blob/main/assets/DungeonRush.png)

## Build
#### To build on linux platforms
Make sure you have met the following dependencies
* SDL2
* SDL2_image
* SDL2_ttf
* SDL2_mixer

**Run the build.sh file contained in the project directory by running the command**
> /bin/bash build.sh
**You will find the binary in bin folder of the project directory**
